Sample #3: Jessica Chan's responsive typography alculation

A 2019 article by Jessica Chan reviews different approaches to responsive typography, all of which feel like dead ends:

  1. Set html { font-size: 100%; }, then set font sizes for various element using rems, and add media queries to deal with different device sizes.

    The h1 on this page was huge when I used that calculation, so I reduced it to h1 {font-size: calc(1rem + 2.5vw)} .

  2. Introduce the calc() function: h1 { font-size: calc(1.3rem + 3.6vw); }

    But, as she notes, there are accessibility issues with viewport units, and media queries will be necessary.
