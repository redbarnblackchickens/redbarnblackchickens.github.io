Sample #3: Jessica Chan's cresponsive typography alculation

This page demonstrates, from a 2019 article by Jessica Chan, an alternative to Heydon's font size calculation. It is:

h1 {
font-size: calc(1.3rem + 3.6vw);
}

And this looks just as bad at large screen widths. Outta control when I expand the page (and too big in general). I can see that clamp() is needed to set an upper limit at large viewports.
