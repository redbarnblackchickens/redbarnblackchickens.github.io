From the clamp() spec, setting width

In dealing with width, clamp() appears to be the tool of choice. Here's an example from the clamp spec:

body { margin: 0 auto; width: min(1000px, calc(70% + 100px)); }

Line length should be around 60 to 75 characters; numbers vary, but this is a decent rule of thumb to start. About 2.5 alphabets is another way to gauge it, so I use the following line as a quick way to judge when the line length is set appropriately, along with viewing how other text on the page appears.
