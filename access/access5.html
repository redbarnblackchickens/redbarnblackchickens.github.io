Another calculation for line length

This article from Piccalilli about using CSS Clamp to create a more flexible wrapper utility (19 January 2021) proposes to set width using clamp() at 90vw, then using what one would previously have specified as the max-width as the clampâ€™s maximum value. The first instance of width is for browsers that don't support clamp().

This page demonstrates their recommendation.

/* wrapper that sets a max width, adds a consistent gutter and horizontally centers the contents */ .wrapper {
width: 90vw;
width: clamp(16rem, 90vw, 70rem);
margin-left: auto;
margin-right: auto;
padding-left: 1.5rem;
padding-right: 1.5rem;
position: relative;
}
