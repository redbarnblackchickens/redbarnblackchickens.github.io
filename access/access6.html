Sampler #6: back to the [complicated] basics

On this page, an attempt to use the clamp() spec as a basis and get something that sort of works for font size, width, and line height:

body {
margin: 0 auto;
width: min(800px, calc(50% + 100px));
}

h1 {
font-size: clamp(1.8rem, 2.5vw, 2.8rem);
}

p {
line-height: 1.5;
font-size: clamp(1rem, 2.25vw, 2rem);
}

code {
line-height: 1.2;
font-size: clamp(.8rem, 2vw, 2rem);
}
